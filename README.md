# User Authentication System (Node.js + JWT)

A secure backend authentication system built using Node.js, Express, MySQL, JWT, and bcrypt. This project provides user registration, login, and protected routes using token-based authentication.

---

## Features

- User registration with password hashing
- Secure login with JWT token generation
- Protected routes using authentication middleware
- MySQL database integration
- RESTful API design

---

## Tech Stack

- Node.js
- Express.js
- MySQL
- JWT (jsonwebtoken)
- bcrypt (password hashing)
